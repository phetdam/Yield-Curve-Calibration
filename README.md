# Yield-Curve-Calibration
Calibrate the Vasicek and possibly the Hull-White model to the US Treasury's par yield curve rates.

# Requirements
The package requirements are as follows:
- pandas: 2.0.3
- matplotlib: 3.7.3

It is also recommended to not use a Python version >= 3.10.8 to avoid an error related to ctypes when
trying to run the script.
